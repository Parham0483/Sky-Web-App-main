Tutorial to completing health check

  1. Each question will have the name of the topic on top
  2. Use the buttons to select a state depending on how you feel around that topic:
    Green: It’s going well and you're happy with that question
    Yellow: There are some concerns but it’s stable
    Red: It’s getting worse and needs urgent help
  3. Use the drop down menu to select a trend for each aspect
  4. You will be able to add comments in any of the three text fields provided
  5. Click continue once done
